data Nat #

Datatype representing the Natural numbers as Zero and Succ(essors) of Zero.

0 = Zero 1 = Succ Zero 2 = Succ (Succ Zero)

natAdd :: Nat -> Nat -> Nat #

Nat + Nat operation.

Accumulates Succ(essors) from first digit onto the other, so this operation is linear on the size of the first digit unless the second is Zero.
